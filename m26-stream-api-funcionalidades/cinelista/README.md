# 🎬 Cinelista

Aplicação desenvolvida com **Next.js** para listar filmes utilizando a API do **The Movie Database (TMDB)**.

O projeto foi desenvolvido durante meus estudos de **Next.js** na EBAC e tem como objetivo praticar conceitos como App Router, rotas dinâmicas, consumo de APIs, Server Components e organização de projetos.

## 🚀 Tecnologias

- Next.js
- React
- TypeScript
- CSS Modules
- Axios
- TMDB API

## ✨ Funcionalidades

- Listagem de filmes
- Página de detalhes do filme
- Consumo de API externa
- Rotas dinâmicas
- Server Components
- Layout responsivo


## ⚙️ Como executar

Clone o repositório:

```bash
git clone https://github.com/gabriellysaless/cinelista.git
```

Entre na pasta do projeto:

```bash
cd cinelista
```

Instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Abra o navegador em:

```
http://localhost:3000
```

## 🔑 Variáveis de ambiente

Crie um arquivo `.env.local` na raiz do projeto utilizando o `.env.example` como base.

Exemplo:

```env
TMDB_API_URL=https://api.themoviedb.org/3
TMDB_API_KEY=SUA_TMDB_API_KEY
NEXT_PUBLIC_TMDB_API_IMG_URL=https://image.tmdb.org/t/p/w500
```
